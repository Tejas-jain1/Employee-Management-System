# Employee Management System

A responsive Employee Management System built using React.js and Tailwind CSS.

## 🚀 Features

- Admin Login
- Employee Login
- Create Task
- Accept Task
- Mark Task as Completed
- Mark Task as Failed
- Live Task Count Updates
- Local Storage Data Persistence
- Responsive Design

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Context API
- Local Storage

## 📂 Installation

```bash
git clone https://github.com/Tejas-jain1/Employee-Management-System.git
```

```bash
cd Employee-Management-System
```

```bash
npm install
```

```bash
npm run dev
```
